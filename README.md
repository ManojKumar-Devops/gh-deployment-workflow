# GitHub Pages Deployment using GitHub Actions

This project demonstrates CI/CD using GitHub Actions.

## 🚀 Features
- Automatic deployment to GitHub Pages
- Trigger only when `index.html` changes

## Project URL

https://ManojKumar-Devops.github.io/gh-deployment-workflow/

## ⚙️ How it works
- Push changes to `main`
- Workflow triggers only if `index.html` changes
- GitHub Actions deploys to GitHub Pages
